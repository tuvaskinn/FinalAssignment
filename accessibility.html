While creating the website

While creating the website, there were a few things you need to keep in mind while coding to ensure a universally designed website. Through the process there were different thing to implement.

Layout

When creating wireframes for how we wanted our website to look, we made sure that the pages was clean. We wanted it to be intuitive and easy to manage. This makes it easier for everyone in any age or with any prerequisites to use our website.

Structure

After deciding on the look for our website, it was time to code it. Most of the aspects to create a universally designed website is done in the HTML code, where the structure of a website is created. Here we made sure to use plenty of semantic tags. By adding semantic tags to your document, you provide additional information about that document, which aids in communication.

3 step self check

After creating the website, we started our accessibility check with a 4 step check on our code checking for:

  1. Check alt text for images and other non-text content
  2. Check color contrast
  3. Make sure your site can be zoomed without loss of content or functionality
  4. Make sure you use plenty of appropriate semantic tags

These four test will make sure that the website is compatible for screenreaders and people with bad vision can still read the page.

Automated test

After our self check, we did an automated test on the website siteimprove.com. On this test we got a full score!

This test checked the universal design of our website, checking for the following problems:

  • Image without alt attribute
  • Picture link is missing alt text attribute
  • The input field has no description
  • The text area has no description
  • There is no description in the drop-down menu
  • The website has several or no title elements
  • Title of the website is missing text
  • The headline is missing text
  • Alternative text for the image is a file name
  • Element ID is not unique
  • The form control label is missing text
  • Use of automatic redirection after a time delay
  • The language of the page is not specified
  • WAI-ARIA image is missing alternate text
